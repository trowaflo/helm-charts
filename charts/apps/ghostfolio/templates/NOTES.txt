IMPORTANT - Secret Requirements:
This chart requires pre-existing secrets. Ensure the following secrets exist:

1. Ghostfolio Application Secrets ({{ .Values.ghostfolio.secretName }}):
   - ACCESS_TOKEN_SALT: Random string for token salt
   - JWT_SECRET_KEY: Random string for JWT signing

2. Database Connection Secret ({{ .Values.ghostfolio.database.secretName }}):
   - {{ .Values.ghostfolio.database.secretKey }}: PostgreSQL connection string
     Format: postgresql://username:password@host:port/database

3. Redis Password Secret ({{ .Values.ghostfolio.redis.passwordSecretName }}):
   - {{ .Values.ghostfolio.redis.passwordSecretKey }}: Redis password

{{- if .Values.postgresql.enabled }}

4. PostgreSQL Bootstrap Secret ({{ .Values.postgresql.cluster.bootstrap.initdb.secret.name }}):
   - username: Database username
   - password: Database password
{{- end }}

Example secret creation:
  kubectl create secret generic {{ .Values.ghostfolio.secretName }} \
    --from-literal=ACCESS_TOKEN_SALT=$(openssl rand -base64 32) \
    --from-literal=JWT_SECRET_KEY=$(openssl rand -base64 32) \
    -n {{ .Release.Namespace }}
